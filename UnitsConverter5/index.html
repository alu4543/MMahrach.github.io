Temperature Conversion Formula

From To Formula
Fahrenheit Celsius ºC = ( ºF - 32) / 1.8
Celsius Fahrenheit ºF = ºC x 1.8 + 32
Fahrenheit Kelvin K = ( ºF + 459.67) / 1.8
Kelvin Fahrenheit ºF = K × 1.8 - 459.67
Celsius Kelvin K = ºC + 273.15
Kelvin Celsius ºC = K – 273.15

Meter Conversion Formula

From To Formula
Meter kilometer 1 km = 1000 m
kilometer Meter 1 m = 0.001 Km
Meter Centimeter 1 cm = 0.01 m
Centimeter Meter 1m = 100 cm
kilometer Centimeter 1 cm = 0.00001 km
Centimeter kilometer 1 km =100000 cm

5.Instalaciones necesarias

Ya la máquina tiene instalados NodeJS y git. Pero con los siguientes comandos instalaremos desde zero todo lo que necesitamos para desplegar nuestra pagina ya que no está instalado todo lo que nos hace falta.

$ sudo apt-get update
												$ sudo apt-get install node
												$ sudo apt-get install nodejs
												$ sudo apt-get install nodejs-legacy
												$ sudo apt-get install npm
												$ sudo apt-get install git

Para comprobar las versiones instaladas

$ node --version
												$ nodejs --version
												$ npm --version
												$ git --version

media

6. git clone

Para desplegamos la práctica anterior en esta maquina tenemos que clonar el repositorio git del proyecto para posteriormente desplegarlo en la máquina.

$ git clone git@github.com:alu4543/STW-5.git

Instalamos las dependencias necesarias de nuestra aplicación, especificadas en el package.json con el comando:
